# 🎓 EduSphere – Personalized Academic Growth Hub

EduSphere is a student-centric academic management platform developed to help students organize, plan, and monitor their academic activities in one place. The system provides a personalized workspace where students can manage tasks, store study resources, prepare for examinations, and track their academic progress through an intuitive dashboard.

The platform also includes an administrator portal that allows the administrator to view registered students and review feedback submitted by students.

---

# 📖 Project Overview

EduSphere was developed as a Full Stack Web Application to simplify academic planning and improve student productivity. Instead of using multiple applications for task management, study planning, notes, and progress tracking, EduSphere brings these features together into a single centralized platform.

The project focuses on improving academic organization through a clean interface, secure authentication, and real-time data management.

---

# ✨ Features

## Student Module

* Student Registration & Login
* Secure Authentication using JWT
* Student Onboarding
* Command Center Dashboard
* Mission Control (Task Management)
* Knowledge Vault (Learning Resources)
* StudyPath (Study Planner)
* RapidPrep (Exam Preparation)
* ProgressIQ (Academic Progress Analytics)
* Feedback Submission
* Profile Management

## Administrator Module

* Secure Admin Login
* View Registered Students
* View Student Feedback
* Review Feedback Status
